# My Class Notes

#### Keyboard shortcuts (VS Code)

| Function | Keyboard Shortcut |
| ---------|---------|
| Indentation | CTRL + K, CTRL + D |
| Suggestions | CTRL + .|
| Delete Line | CTRL + SHIFT + L |
| Move Selected Lines of Code | ALT + ArrowKeys |
| Comment | CTRL + K + C |
| Uncomment | CTRL + K + U |
| Toggle Sidebar | CTRL + B |
| Multi-Select Cursor | CTRL + D |
